## üìù How to Take Screenshots

### Option 1: Browser DevTools
1. Open the application in Chrome/Edge
2. Press F12 to open DevTools
3. Click the device toolbar icon (Ctrl+Shift+M)
4. Set viewport to 1920x1080
5. Use browser's screenshot tool or press Ctrl+Shift+P and type "screenshot"

### Option 2: Screenshot Tools
- **Windows**: Snipping Tool or Snip & Sketch (Win+Shift+S)
- **Mac**: Cmd+Shift+4
- **Linux**: Flameshot or GNOME Screenshot

### Option 3: Browser Extensions
- **Awesome Screenshot** - Full page capture
- **Nimbus Screenshot** - Annotate and edit
- **GoFullPage** - Full page screenshots

## üé® Screenshot Best Practices

### Quality
- Use high resolution (at least 1200px wide)
- Ensure text is readable
- Use good lighting (if capturing screen)
- Avoid glare and reflections

### Content
- Show real data (not empty states)
- Include multiple examples when relevant
- Demonstrate key features clearly
- Use dark mode for some, light mode for others

### Editing
- Crop to remove unnecessary browser chrome
- Add subtle drop shadows for polish
- Annotate key features if needed
- Compress images for web (use TinyPNG or similar)
